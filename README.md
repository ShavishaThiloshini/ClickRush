# 🎮 Click Rush ⚡

A fast-paced, neon-styled **mini browser game** built with pure **HTML, CSS, and Vanilla JavaScript**.  
Test your reflexes, chase high scores, and enjoy quick arcade-style fun right in your browser!

👉 **Play Live:**(https://shavishathiloshini.github.io/ClickRush/)

---

## 🚀 Features
- ⏱️ 30-second fast-action gameplay
- 🎯 Random clickable targets
- 🔥 Combo streak & score multiplier
- 🏆 High score saved using `localStorage`
- 🎵 Click sound effects
- 🌈 Smooth animations & neon UI
- 📱 Fully responsive (desktop & mobile)

---

## 🕹️ How to Play
1. Click **Start Game**
2. Targets will appear randomly on the screen
3. Click them **before they disappear**
4. Build combos for higher scores
5. Miss a target and lose points
6. Beat your **highest score**!

---

## 🛠️ Built With
- **HTML5** – Structure
- **CSS3** – Animations & Neon Styling
- **JavaScript (ES6)** – Game Logic

_No frameworks. No libraries. Just pure web fun._ 😄

## 🌍 Deployment
This project is deployed using **GitHub Pages**.

To deploy your own version:
1. Fork or clone this repository
2. Upload files to a GitHub repository
3. Go to **Settings → Pages**
4. Select:
   - Branch: `main`
   - Folder: `/root`
5. Save and enjoy your live game 🎉
